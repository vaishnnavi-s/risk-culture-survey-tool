# Risk Culture Survey Tool

## Project Overview

Risk Culture Survey Tool is a web-based application designed to evaluate organizational risk awareness using surveys, analytics, and reporting.

## Technology Stack

* Backend: Java Spring Boot
* Frontend: React (Planned)
* Database: PostgreSQL
* Cache: Redis
* Containerization: Docker
